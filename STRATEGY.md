# ADX 自适应永续 ETH 4H 策略

> 主力策略。Donchian breakout + ADX-scaled ATR% stops + 波动率退出。
> 回测代码：`code/backtests/adx_adaptive_perp_eth_4h.py`

## 策略逻辑

```
ADX > 30  → 趋势模式（2.5x ATR% trailing stop, cap 12%）
ADX ≤ 30  → 过渡模式（0.8x ATR% trailing stop, cap 5%）
ATR% > 3.5% → 波动率退出（主动平仓）

双向 Donchian 20 周期突破入场 + 反向突破离场。
空头附加 SMA100 过滤（仅在价格 ≤ SMA100 时做空）。
```

### 核心机制

1. **ADX 体制切换** — Wilder ADX 作为市场状态分类器
   - ADX > 30 = 趋势状态 → 宽松止损捕捉趋势
   - ADX ≤ 30 = 过渡状态 → 紧凑止损控制震荡损耗
2. **ATR% 无量纲止损** — ATR/close×100，跨价格等级可比
3. **波动率退出** — ATR% 超 P97 阈值时强制平仓，83% 优于持有
4. **熔断** — 连续 5 笔亏损 → 暂停交易 24 个 bar

## 已知限制

- **仅 4h ETH 验证通过** — 1h 噪音大回撤深，日线样本不足
- **区间震荡最差** — 假突破反复止损（2026-02→04 期间 -38% vs B&H +3%）
- **深回撤不可消除** — 每年一次 -38%~-56% DD，策略 DNA
- **过渡模式胜率 36%** — 0.8x 止损保本但不产 alpha
- **仅 ETH 全验证通过** — BTC/SOL/BNB/ADA 均正收益但超额未跑赢 B&H

## 已测试并拒绝的改进

- SMA200 趋势过滤 — Sharpe 从 1.19 腰斩至 0.55
