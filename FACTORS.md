# 因子定义

> 所有因子自建，零外部依赖。回测代码：`code/strategies/`

## 主力因子（策略中已使用）

### 1. ADX (Average Directional Index)

Wilder 平滑，14 周期。0-100 刻度，用于体制分类。

| 参数 | 值 |
|------|-----|
| 周期 | 14 |
| 平滑 | Wilder (EMA alpha=1/N) |
| 趋势阈值 | > 30 |
| 过渡阈值 | ≤ 30 |

### 2. Donchian 通道

N 周期最高/最低价突破信号。

| 参数 | 值 |
|------|-----|
| 多头周期 | 20 |
| 空头周期 | 20 / 8（激进版） |

### 3. ATR% (Average True Range 百分比化)

ATR / close × 100。无量纲止损基准。

| 参数 | 值 |
|------|-----|
| 周期 | 14 |
| 趋势倍数 | 2.5x, cap 12% |
| 过渡倍数 | 0.8x, cap 5% |
| 波动率退出 | ATR% > 3.5% (P97) |

### 4. SMA100

趋势过滤。空头仅在价格 ≤ SMA100 时开仓。
